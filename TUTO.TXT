EXECUTER LE PROGRAMME :
1) ALLER DANS L'ARCHIVE BUILD VIA LE TERMINAL
2) FAIRE LA COMMANDE : "make start"
3) FAIRE LA COMMANDE : "./start"

EN CAS DE PROBLEME TUTO REFAIRE CMAKE :
1) LANCER CMAKE
2) METTRE POUR SOURCE CODE LE DOSSIER "SEANCE 2"
3) METTRE POUR BUILD LE DOSSIER "BUILD" de SEANCE 2
4) CLIQUER SUR "CONFIGURE"
5) CLIQUER SUR "GENERATE"

OU TROUVER LES FICHIERS FEN :
1) ALLER DANS LE DOSSIER "SEANCE 2"
2) ALLER DANS LE DOSSIER "build"
3) ALLER DANS LE DOSSIER "FEN"
4) LES FICHIERS SONT LA

LISTE DES BUGS :
1) ON NE PEUT PAS ECRIRE AVEC WRITE_FEN DANS UN FICHIER QUI EST DEJA ECRIE
